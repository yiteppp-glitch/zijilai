三步开始

一分钟用起来

1

装上扩展

解压后在 chrome://extensions 打开开发者模式，「加载已解压」选该文件夹。

2

填好 Key

打开设置，选「自填接口」填你自己的 OpenAI 兼容 Base URL + Key + 模型（需支持函数调用）。

3

开聊

点工具栏图标或按 ⌘⇧K 打开侧边栏，直接说「总结这个页面」「整理我的标签」。

关于安全

它在你手里，钥匙也是

  • 本地对话、会话历史、设置只存在你的浏览器；网页内容只发往你配置的模型接口，没有中间服务器。
  • 确认跳转 / 填写 / 点击 / 开关标签等会改动状态的动作，会先弹确认，你点允许才执行（范围可调）。
  • 隔离第三方 MCP 工具的名称与返回内容一律当作外部数据，绝不会被当成指令执行。
  • 自带 Key公开版不内置任何密钥，每个人填自己的 Key——钥匙只在你本地。
